## Test Project

### Create Cordova Project

[Follow the cordova guidelines for creating a project](https://cordova.apache.org/ "Cordova Guide")

* cordova platform add android

* cordova build android

* Download plugin from bitbucket to computer

* Add plugin

  * cordova plugin add C:\path_to_downloaded_plugin\RowPlux

  * replace www folder in the created cordova project with the www folder in Sample folder of downloaded plugin

* cordova prepare android

* cordova run android
